    üßë‚Äçüíª Account¶

    • Login ‚Üí Numeric ID of your trading account.
    • Password ‚Üí Investor or trader password. Grants access to terminal.
    • Server ‚Üí Broker server name (e.g., RoboForex-Demo).
    • Balance ‚Üí Money currently on the account.
    • Equity ‚Üí Balance + open positions PnL.
    • Margin ‚Üí Funds locked for open positions.
    • Free Margin ‚Üí Equity ‚àí Margin.
    • Leverage ‚Üí Ratio (e.g., 1:500) showing how much borrowed funds you can use.

    üìà Market Info¶

    • Symbol ‚Üí Instrument identifier (e.g., EURUSD).
    • Quote ‚Üí Current bid/ask prices for a symbol.
    • Digits ‚Üí Decimal precision of the quote (e.g., 5 digits ‚Üí 1.23456).
    • Point ‚Üí Smallest price step for the symbol (e.g., 0.00001 for EURUSD).
    • Lot ‚Üí Standard trade size (usually 100,000 units base currency).
    • Lot Step ‚Üí Minimum increment allowed when specifying volume.
    • Contract Size ‚Üí Amount of base currency per lot.
    • Stops Level ‚Üí Minimum distance (in points) required for SL/TP from current price.

    üì¶ Orders¶

    • Order ‚Üí Instruction to buy or sell a symbol.
    • Market Order ‚Üí Executed immediately at current market price.
    • Pending Order ‚Üí Placed to execute in the future at a specific price (Limit or Stop).
    • Ticket ‚Üí Unique ID of an order (int64).
    • SL (Stop Loss) ‚Üí Protective level to cap loss.
    • TP (Take Profit) ‚Üí Target level to close with profit.
    • Magic Number ‚Üí User-defined integer to tag EAs/orders.
    • Comment ‚Üí Free text attached to an order.

    üîÑ Order Types (MT4)¶

    • OP_BUY ‚Üí Buy at market.
    • OP_SELL ‚Üí Sell at market.
    • OP_BUYLIMIT ‚Üí Pending: buy if price drops to X.
    • OP_SELLLIMIT ‚Üí Pending: sell if price rises to X.
    • OP_BUYSTOP ‚Üí Pending: buy if price rises to X.
    • OP_SELLSTOP ‚Üí Pending: sell if price drops to X.

    üîå Connection & RPC¶

    • gRPC ‚Üí Protocol used by GoMT4 to expose MT4 functions.
    • Port 50051 ‚Üí Default local address where server listens (127.0.0.1:50051).
    • Stream ‚Üí Long‚Äëlived connection pushing updates (quotes, orders, history).
    • Unary RPC ‚Üí One‚Äëshot request/response (e.g., GetQuote).

    üìä History & Streaming¶

    • Quote History ‚Üí Past bid/ask points (OHLC bars or ticks).
    • Order History ‚Üí List of closed trades for a period.
    • StreamQuotes ‚Üí Continuous tick updates.
    • StreamTradeUpdates ‚Üí Real-time feed of order lifecycle events.

    üõ°Ô∏è Errors & Codes¶

    • MrpcError ‚Üí Generic RPC error (code + message).
    • OrderError ‚Üí Error tied to a specific order (invalid volume, price, etc.).
    • Slippage ‚Üí Max price deviation allowed when executing orders.
    • Requote ‚Üí Server rejects trade at requested price, offers new one.
